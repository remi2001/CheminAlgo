Pour modifier les points qui sont obligatoires, il faut ouvrir le fichier Point.csv .
La signafication des valeurs : 
CoordonnéeX;CoordonnéeY;NombreDePointRapporté;Obligatoire

Il suffit de modifier le 0 en 1 pour que le point devienne obligatoire. A savoir, il y les points obligatoires et secondaires. Ils sont différenciable grâce au nombre de point qu'il rapporte. Les points qui n'apportent pas 30 points sont les points de type secondaires. Les autres, sont les points de type obligatoires.

Pour modifier la carte, il faut ouvrir le fichier Map.csv . Il faut absolument avoir 20 valeurs en ligne et en colonne.
